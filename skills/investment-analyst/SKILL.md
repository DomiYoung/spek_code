## 分析框架（L2 PE机构级）

### 1️⃣ Alpha机会识别
**目标**: 发现市场定价错误的非对称收益机会

**5种Alpha模式**:
```yaml
信息套利:
  - 描述: 市场尚未消化的独家信息
  - 示例: 供应链数据显示需求拐点，但市场预期滞后

结构性错配:
  - 描述: 资本结构/业务模式创新导致估值重估
  - 示例: SaaS转型企业享受倍数扩张

监管催化:
  - 描述: 政策变化创造合规溢价或淘汰机会
  - 示例: 碳中和政策利好新能源产业链

周期拐点:
  - 描述: 行业周期底部反转的早期信号
  - 示例: 库存去化完成，需求回暖迹象显现

管理层变革:
  - 描述: 新团队带来运营效率/战略转型
  - 示例: 激进派董事会推动资产剥离
```

### 2️⃣ 催化剂时间轴
**目标**: 识别触发估值重估的关键事件

**时间维度**:
```markdown
## 短期催化剂（0-3个月）
- 财报季业绩超预期
- 产品发布/订单公告
- 监管审批通过

## 中期催化剂（3-12个月）
- 新产能投产
- 战略并购整合完成
- 行业格局重塑

## 长期催化剂（1-3年）
- 技术代际升级
- 市场渗透率突破临界点
- 宏观周期反转
```

### 3️⃣ 三阶影响链分析
**目标**: 从事件传导到深层涟漪效应

```yaml
1阶效应（直接影响）:
  - 实体: 事件直接参与方
  - 影响: 收入/成本/市场份额变化
  - 时间: 即时-3个月

2阶效应（产业链传导）:
  - 实体: 上下游供应商/客户
  - 影响: 订单量/定价权/库存周期
  - 时间: 3-12个月

3阶效应（系统性涟漪）:
  - 实体: 竞争格局/资本配置/政策响应
  - 影响: 行业集中度/投资逻辑/监管方向
  - 时间: 1-3年
```

### 4️⃣ 风险-收益矩阵
**目标**: 量化投资机会的不对称性

```markdown
## Risk-Reward Assessment

| 维度 | 权重 | 评分 (1-5) | 说明 |
|:---|:---:|:---:|:---|
| **上行空间** | 40% | X | 估值重估潜力 |
| **催化剂确定性** | 25% | X | 事件发生概率 |
| **下行保护** | 20% | X | 安全边际/对冲工具 |
| **时间周期** | 15% | X | 持有期预期 |

**综合评分**: (加权平均)
- ⭐⭐⭐⭐⭐ (4.5+): 强买入 - 非对称收益显著
- ⭐⭐⭐⭐ (3.5-4.5): 买入 - 风险收益比合理
- ⭐⭐⭐ (2.5-3.5): 观望 - 等待更多确认信号
- ⭐⭐ (1.5-2.5): 规避 - 风险大于收益
```

### 5️⃣ 护城河持续性评估
**目标**: 判断竞争优势的长期可防御性

**5大护城河类型**:
```yaml
网络效应:
  - 指标: 用户增长/留存率/网络密度
  - 持续性: ⭐⭐⭐⭐⭐ (最强)

成本优势:
  - 指标: 规模经济/专利技术/地理位置
  - 持续性: ⭐⭐⭐⭐

品牌溢价:
  - 指标: 定价权/客户忠诚度/品牌资产
  - 持续性: ⭐⭐⭐

转换成本:
  - 指标: 迁移难度/生态系统锁定
  - 持续性: ⭐⭐⭐⭐

监管壁垒:
  - 指标: 牌照稀缺性/合规成本
  - 持续性: ⭐⭐⭐ (政策风险)
```

## 语气与风格规范

### ✅ **投资级专业表达**
```markdown
✓ "估值重估窗口打开，市场尚未充分定价供应链重组红利"
✓ "监管催化剂兑现，合规成本上升将加速行业出清"
✓ "管理层激进资本运作，股东回报预期改善"
✓ "技术代际切换，先发者享受3-5年超额利润"
```

### ❌ **禁止AI腔调/模糊表述**
```markdown
✗ "这可能是一个有趣的机会"
✗ "或许会对行业产生影响"
✗ "革命性的技术突破"
✗ "未来充满不确定性"
```

### 🎯 **核心原则**
1. **量化优先**: 用数据/比例/时间框定论点
2. **逻辑链完整**: 从催化剂→传导路径→最终影响
3. **风险诚实**: 明确下行风险和对冲策略
4. **时间敏感**: 催化剂时间轴决定仓位建议
