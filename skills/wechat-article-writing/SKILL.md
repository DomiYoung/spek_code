# 微信公众号文章写作

帮助创作「Domi的AI生活信札」风格的公众号文章：口语化表达+深度思辨，真实经历+数据支撑。

## 写作工作流

```text
选题确认 → 素材收集 → 大纲设计 → 分节撰写 → 金句提炼 → 结尾互动
```

## 核心风格

### 标题公式

- 数字+教训：「2000刀买来的教训：我给AI打了一年工」
- 反差感：看似负面实则有价值
- 避免：标题党、夸大、悬疑式

### 开篇模式

```markdown
我是Domi，一个前端开发。

[年份]，我"死"了很多次。
[或其他钩子句]
```

- 3句内建立身份认同
- 用钩子制造阅读期待

### 章节结构

```markdown
## 01｜[章节标题]

[叙事段落：真实经历/场景]

[分析段落：为什么会这样]

**[金句总结：加粗，一句话提炼]**

---
```

- 用 `01｜02｜` 格式编号
- 每节 200-400 字
- 以 `---` 分隔章节

### 语言风格

| 要素   | 做法   | 示例                                                     |
| ------ | ------ | -------------------------------------------------------- |
| 口语化 | 像聊天 | "你会为了那一点点，修到凌晨"                             |
| 短句   | ≤20字  | "我没死成。"                                             |
| 金句   | 加粗   | **工具本该是手的延伸。现在，手成了工具的延伸。**         |
| 引用   | 用 `>` | > 45%的人花更多时间debug                                 |
| 列表   | 用 `•` | • 写代码稳不稳？                                         |

### 结尾模板

```markdown
## 写在最后

[1-2段总结性思考]

**[核心观点/人生态度]**

---

如果你也[经历过xxx]——

**[互动问题1]**
**[互动问题2]**

评论区聊聊。

---

**Domi的AI生活信札｜第[N]期**
*像机器一样进化，像人一样生活*
```

## 附录规范

如有消费/数据类内容，添加附录：

```markdown
## 附：我的折腾账单

[具体项目列表]

合计：[总金额]

*以上均为个人真实消费记录，仅供参考。*
```

## 检查清单

发布前验证：

- [ ] 标题吸引但真实
- [ ] 开篇3句内建立钩子
- [ ] 章节用 `01｜` 格式
- [ ] 每节有金句（加粗）
- [ ] 结尾有互动问题
- [ ] 包含 slogan
