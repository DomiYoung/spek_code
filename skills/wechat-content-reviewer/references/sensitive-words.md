# 微信公众号敏感词检测

## 一、平台限流词库

### 绝对化用语（触发降权）
```
最、最好、最佳、最优、最强、最高、最低
第一、唯一、独家、首创、首发
100%、百分百、绝对、完全、必定
```

### 夸张诱导词（触发审核）
```
震惊、惊呆、吓人、恐怖、可怕
必火、必爆、必看、必转、必收藏
速看、紧急、最后、限时、马上删
```

### 敏感话题词（高风险）
```
政治类: 政策解读、官方消息（非官方账号）
金融类: 暴富、躺赚、稳赚、保本、理财推荐
医疗类: 根治、特效、祖传秘方
```

## 二、低质量内容信号

### 标题党特征
```
- "不转不是中国人"
- "看完泪目/泪崩"
- "99%的人不知道"
- "看到最后惊呆了"
- "速度收藏，马上删"
```

### 情绪化表达（降低专业度）
```
- "太厉害了！"
- "绝了绝了"
- "服了服了"
- "YYDS"
- "绷不住了"
```

### 模糊表达（降低信任度）
```
- "这个东西"
- "某某工具"
- "有人说"
- "据说"
- "听说"
```

## 四、替换建议

| 敏感词 | 推荐替换 |
|--------|----------|
| 最好的 | 值得推荐的/我常用的 |
| 100%有效 | 亲测有效/实测好用 |
| 必须看 | 建议收藏/推荐阅读 |
| 震惊 | 发现/注意到 |
| 暴富 | 增收/提升收入 |
| 轻松赚钱 | 副业思路/变现方式 |
| 独家 | 深度/详细 |
| 首发 | 最新/近期 |

## 五、自动检测规则

```python
# 检测逻辑伪代码
def check_sensitive(text):
    risks = []

    # 绝对化检测
    if contains(text, ABSOLUTE_WORDS):
        risks.append(("绝对化用语", "高风险"))

    # 夸张词检测
    if contains(text, EXAGGERATED_WORDS):
        risks.append(("夸张诱导词", "中风险"))

    # 标题党检测
    if matches(text, CLICKBAIT_PATTERNS):
        risks.append(("标题党嫌疑", "中风险"))

    # 敏感话题检测
    if contains(text, SENSITIVE_TOPICS):
        risks.append(("敏感话题", "高风险"))

    return risks
```
