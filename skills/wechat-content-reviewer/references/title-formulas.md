# 微信公众号标题优化公式库

## 高 CTR 标题公式

### 1. 数字+悬念型
```
[数字] + [身份/场景] + [利益/悬念]

示例:
- "3个月用AI写代码，我发现了5个反常识真相"
- "这7个AI工具，让我效率提升300%"
- "90%的人都踩过的3个坑"
```

### 2. 身份共鸣型
```
[身份] + 必读/必看 + [利益]

示例:
- "程序员必读：AI时代的生存法则"
- "运营人必看：这个工具省了我80%时间"
- "创业者必读：别让AI成为你的负债"
```

### 3. 反常识型
```
[常识] + 其实 + [反转]

示例:
- "AI写代码不是更快，而是更慢——如果你这样用"
- "你以为的效率工具，其实是效率杀手"
- "越努力越穷？问题出在这里"
```

### 4. 时效+限定型
```
[时间限定] + [我/你] + [行动/发现]

示例:
- "2024年最后悔没早点知道的AI工具"
- "今年用下来最值的3个付费工具"
- "这个月我靠AI多赚了5000块"
```

### 5. 问句引导型
```
[问题] + [暗示答案]

示例:
- "想用AI开发多个项目搞钱？先解决这个坑"
- "为什么你的AI用不出效果？原因在这"
- "还在手动处理数据？这个方法省90%时间"
```

### 6. 对比冲突型
```
[A情况] vs [B情况] + [结果差异]

示例:
- "同样用AI，为什么他效率是你的10倍"
- "普通人vs高手：AI工具使用差异"
- "用对vs用错：AI编程的两种结局"
```

## 标题黄金区间

| 平台 | 最佳字数 | 说明 |
|------|----------|------|
| 公众号 | 15-25字 | 手机端完整显示 |
| 订阅号 | 18-22字 | 推送列表最佳 |
| 服务号 | 15-20字 | 会话列表显示 |

## CTR 提升元素

| 元素 | 提升效果 | 使用建议 |
|------|----------|----------|
| 数字 | +15-20% | 优先使用奇数 |
| 问句 | +10-15% | 暗示有答案 |
| "你" | +10% | 增加代入感 |
| 时间词 | +12% | 2024年/今年/这个月 |
| 对比词 | +15% | vs/但是/其实 |
| 利益词 | +18% | 省钱/赚钱/效率 |

## 避雷词库

### 平台限流词
```
绝对化: 最、第一、唯一、100%
夸张词: 震惊、惊呆、必火、疯狂
诱导词: 必转、速看、紧急、最后机会
敏感词: 暴富、躺赚、割韭菜
```

### 低质量信号词
```
情绪化: 太厉害了、绝了、服了
模糊化: 这个东西、某某某
标题党: 不转不是中国人、看完泪目
```

## 标题自检清单

```
☐ 字数在 15-25 字之间
☐ 包含至少一个数字或问句
☐ 有明确的目标读者身份
☐ 传递清晰的阅读价值
☐ 无绝对化/夸张/诱导词
☐ 与正文内容一致
☐ 手机端预览效果良好
```
