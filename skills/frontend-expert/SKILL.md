## 进化记录 (Evolved: 2025-12-24)

### 组件可测试性
- **显式类名规则**：每个可交互组件必须有一个稳定的、语义化的 CSS 类名（如 `.intelligence-card`, `.nav-item`）
- 这些类名不仅用于样式，更用于自动化测试选择器
- 避免仅依赖动态生成的类名（如 Tailwind 的 `p-4 bg-white` 组合）

### 示例
```tsx
// ❌ 不可测试
<div className="bg-white p-4 rounded-lg">

// ✅ 可测试
<div className="intelligence-card bg-white p-4 rounded-lg">
```
